## Usage
### Files
```
wget -q -O - https://raw.githubusercontent.com/udunuss/dotfiles/main/getdotfiles | /bin/bash
```
### Packages
```
wget -q -O - https://raw.githubusercontent.com/udunuss/dotfiles/main/pkgs | /bin/bash
```
